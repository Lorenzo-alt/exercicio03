Colégio Tecnico Madeira

O Colégio Tecnico Madeira tem o foco na formação de profissionais Tecnólogos. No colégio o ensino segue um foco em Tecnologia, uma área que os numeros de vagas para profissionais qualificados ainda é muito superior a formação de individuos. É de total interesse de nossa intituição permitir que nossos alunos tenham acesso as principais tecnologias da área e que seu apreendizado seja focado na prática

Matérias Técnicas

LINGUAGEM DE PROGRAMAÇÃO EM C

Um curso com a intenção de demonstrar toda lógica por traz de um sistema, abordado em linguagem C visando ser um primeiro passo para os alunos em tecnologia, dando a eles toda experiencia de aprender uma linguagem completa

WEB DESIGN

O curso de Web Design visa ensinar HTML 5 e CSS 3 completo

DESENVOLVIMENTO DE GAMES

Curso que abrange um desenvolvimento de jogos em Unity para iniciantes

BANCO DE DADOS

Um curso em MySql ensinando a montar um banco completo e como alimenta-lo

COMPUTAÇÃO GRÁFICA

Curso de modelagem 3D para uso em computação grafica em todas a vertentes
